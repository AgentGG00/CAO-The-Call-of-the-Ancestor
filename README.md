Foundry-VTT-Modul für die homebrew Ancestor-Armbrust inkl. Magazin-Logik, Statuseffekten und einem eigenen Crafting-System.

## Features
- Ancestor-Armbrust mit Magazin-System (Standard, Kälte, Blitz, Gift)
- Automatisierte Schadens- und Status-Logik pro Munitionstyp
- Crafting-System (Rezepte, Zutaten, Cauldron)
- Mehrsprachig (EN, DE, FR, ES, IT, RU, JA, CN)
